# Gligers' vrchat-osc-scripts

> A set of scripts to interact with VRChat via OSC,
> currently supporting sending a shock to the [PiShock API](https://www.pishock.com/) when the tail grab is released.

The documentation is split into several pages. If you want to:
- Install and use it with the executable (simplest way) - follow [installation using executable](docs/installation-executable.md)
- Install it using Python (advanced level) - follow [installation using python](docs/installation-python.md)
- Prepare your avatar, follow [installation from Unity side](docs/installation-unity.md)
- Configure the script, follow [configuration](docs/configuration.md)
